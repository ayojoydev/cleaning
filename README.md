# Клининговая компания — учебный клиент-серверный проект

Небольшой учебный проект для курсовой работы: система для клининговой компании.  
Состоит из двух независимых частей:

- **Серверное приложение** на Java  
- **Клиентское приложение** на Go  

Обмен данными происходит по HTTP (IP + порт), в дальнейшем планируется JSON и работа с БД PostgreSQL.

---

## Технологии

**Сервер:**

- Java 17+
- Maven
- Встроенный HTTP-сервер (`com.sun.net.httpserver.HttpServer`) на этапе прототипа
- Планируется: PostgreSQL + JDBC, MVC, логирование, внешний конфиг

**Клиент:**

- Go (CLI-приложение)
- `net/http` для запросов к серверу
